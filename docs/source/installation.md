# Installation

## Using pip

Install cligram from PyPI:

\`\`\`bash
pip install cligram
\`\`\`

## With optional dependencies

For performance optimizations:

\`\`\`bash
pip install cligram[speedups]
\`\`\`

For media handling:

\`\`\`bash
pip install cligram[media]
\`\`\`

For all extras:

\`\`\`bash
pip install cligram[speedups,media]
\`\`\`

## From source

\`\`\`bash
git clone https://github.com/Aeliux/cligram.git
cd cligram
poetry install
\`\`\`
